## About This Project

This project was developed to practice and strengthen my SQL skills by designing and implementing a complete relational database for an E-Commerce Management System.

The database schema, relationships, constraints, and overall design were created as part of my SQL learning journey. The dataset is based on an e-commerce dataset from Kaggle, and missing sample records were generated with the assistance of ChatGPT to create a complete and consistent database for practice.

This project covers:

- Database Design
- Relational Modeling
- Primary & Foreign Keys
- SQL Constraints
- Data Integrity
- Sample Data Management

> **Note:** This project is intended for educational and portfolio purposes.


##  Dataset

- Original dataset inspired by **Kaggle E-Commerce Dataset**
- Missing sample data was generated with the assistance of **ChatGPT** to complete the database while maintaining relational consistency.





##  Future Scope

This project is the database foundation of a complete E-Commerce Management System. In future updates, the following features will be implemented:

- Develop the backend using **Django** and Django REST Framework.
- Build the frontend using **HTML**, **CSS**, and **JavaScript**.
- Implement user authentication and authorization.
- Product browsing and search functionality.
- Shopping cart management.
- Order placement and order history.
- Payment management and order tracking.
- Admin dashboard for managing customers, products, orders, and payments.

- Responsive user interface for desktop and mobile devices.
